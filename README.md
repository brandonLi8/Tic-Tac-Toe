Tic Tac Toe
=======
By Brandon Li, done completely in tkinter
## features
- ai system
- scores
- help mode
- restart system

## Prerequisites
- python (i recommend 3.6.1 or higher )
- any python interpretor

## how to use
1. download into file
2. open TicTacToe.py and save it into the folder 
3. run TicTacToe.py
